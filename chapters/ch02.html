    Chapter 02

    資料讀取與繪圖

    使用 readr、readxl、Base R 與 ggplot2 讀取、整理及呈現資料

    1. 讀取 CSV

    Base R

    df <- read.csv(
  "data.csv",
  fileEncoding = "UTF-8"
)

head(df)

    readr

    library(readr)

df <- read_csv("data.csv")

head(df)
    中文亂碼時，可依檔案編碼嘗試 UTF-8、UTF-8-BOM 或 Big5。

    2. 讀取 Excel

    install.packages("readxl")
library(readxl)

df <- read_excel(
  "data.xlsx",
  sheet = "Sheet1"
)

head(df)

    sheet 可以填入工作表名稱，也可使用數字指定工作表位置。

    3. 初步檢查資料

    head(df)       # 前六筆
tail(df)       # 後六筆
dim(df)        # 列數與欄數
names(df)      # 欄位名稱
str(df)        # 資料結構
summary(df)    # 摘要統計
    剛讀入資料時，至少先執行 head()、str() 與 summary()。

    4. 描述統計

    平均數（Mean）

    mean(df$score, na.rm = TRUE)

    平均數為所有觀察值加總後，再除以觀察值的數量。

    中位數（Median）

    median(df$score, na.rm = TRUE)

    中位數是資料排序後位於中間的數值，較不容易受到極端值影響。

    眾數（Mode）

    R 沒有直接計算統計眾數的內建函數，因此可以自行建立函數： 

    mode_value <- function(x) {
  x <- x[!is.na(x)]
  unique_values <- unique(x)

  unique_values[
    which.max(tabulate(match(x, unique_values)))
  ]
}

mode_value(df$score)
    平均數適合描述整體平均水準；資料存在極端值時，可搭配中位數一起觀察； 眾數則可找出資料中最常出現的數值或類別。

    5. 使用 Base R 繪圖

    折線圖

    x <- c(1, 2, 3, 4, 5)
y <- c(10, 20, 15, 25, 30)

plot(
  x,
  y,
  type = "l",
  main = "Line Plot",
  xlab = "X",
  ylab = "Y"
)

    長條圖

    categories <- c("A", "B", "C", "D")
values <- c(23, 45, 12, 36)

barplot(
  values,
  names.arg = categories,
  main = "Bar Plot",
  xlab = "Category",
  ylab = "Value"
)

    直方圖

    set.seed(123)

values <- rnorm(1000)

hist(
  values,
  breaks = 30,
  main = "Histogram",
  xlab = "Value"
)

    散佈圖

    set.seed(123)

x <- runif(50)
y <- runif(50)

plot(
  x,
  y,
  main = "Scatter Plot",
  xlab = "X",
  ylab = "Y"
)

    盒鬚圖（Boxplot）

    set.seed(123)

values <- rnorm(100)

boxplot(
  values,
  main = "Boxplot",
  ylab = "Value",
  col = "lightblue"
)

    盒鬚圖可快速觀察資料的中位數、四分位數、資料分散程度與可能的離群值。

    6. 使用 ggplot2

    library(ggplot2)

    直方圖

    ggplot(df, aes(x = column_name)) +
  geom_histogram(bins = 30) +
  labs(
    title = "Histogram",
    x = "Value",
    y = "Frequency"
  )

    依類別分組的散佈圖

    ggplot(
  df,
  aes(
    x = col1,
    y = col2,
    color = category_column
  )
) +
  geom_point()

    盒鬚圖

    ggplot(
  df,
  aes(
    x = category_column,
    y = value_column
  )
) +
  geom_boxplot() +
  labs(
    title = "Boxplot",
    x = "Category",
    y = "Value"
  )
    ggplot2 的 color 可依類別欄位分組上色； 盒鬚圖則適合比較不同類別的資料分布。

    7. 儲存圖表

    plot_object <- ggplot(
  df,
  aes(
    x = col1,
    y = col2
  )
) +
  geom_point()

ggsave(
  "scatter_plot.png",
  plot = plot_object,
  width = 8,
  height = 6
)

    本章重點整理

    • CSV 可使用 read.csv() 或 read_csv() 讀取。
    • Excel 可使用 read_excel() 讀取。
    • 資料讀入後應先利用 head()、str() 與 summary() 檢查資料。
    • 可利用平均數、中位數與眾數描述資料的集中趨勢。
    • Base R 可以繪製折線圖、長條圖、直方圖、散佈圖與盒鬚圖。
    • ggplot2 可建立容易調整與擴充的統計圖表。

    練習題

    1. 讀取一份 CSV，並顯示前六筆資料。
    2. 計算一個數值欄位的平均數、中位數與眾數。
    3. 使用 Base R 繪製直方圖與盒鬚圖。
    4. 使用 ggplot2 建立依類別分組的散佈圖。
    5. 使用 ggplot2 比較不同類別的盒鬚圖。
