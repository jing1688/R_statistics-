    Chapter 02

    資料讀取與繪圖

    使用 readr、readxl、Base R 與 ggplot2 讀取及呈現資料

    1. 讀取 CSV

    Base R

    df <- read.csv(
  "data.csv",
  fileEncoding = "UTF-8"
)

head(df)

    readr

    library(readr)

df <- read_csv("data.csv")
head(df)
    中文亂碼時，可依檔案編碼嘗試 UTF-8、UTF-8-BOM 或 Big5。

    2. 讀取 Excel

    install.packages("readxl")
library(readxl)

df <- read_excel(
  "data.xlsx",
  sheet = "Sheet1"
)

head(df)

    sheet 可以填工作表名稱，也可用數字指定位置。

    3. 初步檢查資料

    head(df)       # 前六筆
tail(df)       # 後六筆
dim(df)        # 列數與欄數
names(df)      # 欄位名稱
str(df)        # 資料結構
summary(df)    # 摘要統計
    剛讀入資料時，至少先執行 head()、str() 與 summary()。

    4. 描述統計

    平均數（Mean）

    mean(df$score)

    計算資料的平均值。

    中位數（Median）

    median(df$score)

    取得資料排序後位於中間的數值，不容易受到極端值影響。

    眾數（Mode）

    R 沒有內建眾數函數，可自行建立：

    mode_value <- function(x){
  ux <- unique(x)
  ux[which.max(tabulate(match(x, ux)))]
}

mode_value(df$score)
    平均數適合一般資料；若資料存在極端值，可搭配中位數一起觀察；眾數適合分析最常出現的類別或數值。

    5. 使用 Base R 繪圖

    折線圖

    x <- c(1, 2, 3, 4, 5)
y <- c(10, 20, 15, 25, 30)

plot(x, y, type = "l",
     main = "Line Plot",
     xlab = "X", ylab = "Y")

    長條圖

    categories <- c("A", "B", "C", "D")
values <- c(23, 45, 12, 36)

barplot(values, names.arg = categories)

    直方圖

    set.seed(123)
data <- rnorm(1000)

hist(data, breaks = 30)

    散佈圖

    set.seed(123)
x <- runif(50)
y <- runif(50)

plot(x, y)

    5. 使用 ggplot2

    library(ggplot2)

    直方圖

    ggplot(df, aes(x = column_name)) +
  geom_histogram(bins = 30) +
  labs(title = "Histogram")

    依類別分組的散佈圖

    ggplot(df, aes(
  x = col1,
  y = col2,
  color = category_column
)) +
  geom_point()
    ggplot2 的 color 可依類別欄位分組上色。

    盒鬚圖（Boxplot）

    set.seed(123)
data <- rnorm(100)

boxplot(
  data,
  main = "Boxplot",
  ylab = "Value",
  col = "lightblue"
)

    盒鬚圖可快速觀察資料的中位數、四分位數、離群值與資料分散程度。

    6. 儲存圖表

    plot_object <- ggplot(df, aes(x = col1, y = col2)) +
  geom_point()

ggsave(
  "scatter_plot.png",
  plot = plot_object,
  width = 8, height = 6
)

    本章重點整理

    • CSV 可使用 read.csv() 或 read_csv()
    • Excel 可使用 read_excel()
    • 資料讀入後先檢查欄位與資料型態
    • Base R 與 ggplot2 都能建立統計圖表

    練習題

    1. 讀取一份 CSV 並顯示前六筆。
    2. 使用 Base R 繪製直方圖。
    3. 使用 ggplot2 建立依類別分組的散佈圖。
