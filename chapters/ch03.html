    Chapter 03

    敘述統計與相關性分析

    使用 R 計算集中趨勢、離散程度、風險指標與變數之間的線性關係

    本章學習目標

    • 複習平均數、中位數與眾數。
    • 理解幾何平均與加權平均的使用時機。
    • 計算全距、變異數、標準差、MAD 與變異係數。
    • 區分樣本統計量與母體統計量。
    • 理解共變異數與相關係數的差異。
    • 能使用 Sharpe Ratio 評估風險調整後報酬。
    • 能繪製散佈圖並加入線性迴歸趨勢線。

    1. 建立範例資料

    本章使用以下兩組資料進行示範。假設 x 為讀書時間，y 為考試成績。 

    x <- c(2, 3, 4, 5, 6, 7, 8, 9)

y <- c(55, 60, 65, 68, 72, 78, 82, 88)

df <- data.frame(
  study_hours = x,
  score = y
)

df

    2. 集中趨勢

    集中趨勢用來描述一組資料的中心位置，常見指標包括平均數、 中位數與眾數。

    平均數（Mean）

    平均數是所有觀察值加總後，再除以觀察值的數量。

    平均數 = 所有觀察值總和 ÷ 觀察值數量 
    scores <- c(60, 65, 70, 70, 75, 80, 90)

mean(scores)

    若資料包含缺失值 NA，可以加入 na.rm = TRUE： 

    scores <- c(60, 65, 70, NA, 75, 80, 90)

mean(scores, na.rm = TRUE)

    中位數（Median）

    中位數是將資料由小到大排列後，位於中間位置的數值。 它較不容易受到極端值影響。 

    median(scores, na.rm = TRUE)

    眾數（Mode）

    眾數是資料中出現次數最多的數值。R 的 mode() 並不是用來計算統計眾數，因此需要自行建立函數。 

    get_mode <- function(x) {
  x <- x[!is.na(x)]

  values <- unique(x)

  values[
    which.max(tabulate(match(x, values)))
  ]
}

scores <- c(60, 65, 70, 70, 75, 80, 90)

get_mode(scores)
    平均數容易受到極端值影響；中位數較穩健；眾數則適合找出最常出現的數值或類別。

    3. 其他平均數

    幾何平均（Geometric Mean）

    幾何平均常用於成長率、報酬率或倍數資料。其概念是將數值相乘後， 再取觀察值數量的次方根。

    幾何平均只能直接套用在正數資料上。 
    growth_rates <- c(1.05, 1.08, 0.97, 1.10)

geometric_mean <- prod(growth_rates)^(1 / length(growth_rates))

geometric_mean

    也可以利用對數進行計算： 

    geometric_mean <- exp(
  mean(log(growth_rates))
)

geometric_mean

    加權平均（Weighted Mean）

    當不同觀察值的重要程度不相同時，可以使用加權平均。

    加權平均 = 各數值乘以權重後的總和 ÷ 權重總和 
    scores <- c(80, 75, 90)

weights <- c(0.3, 0.3, 0.4)

weighted.mean(
  scores,
  weights
)

    例如期中考占 30%、期末考占 30%、報告占 40%，便可利用加權平均計算總成績。

    4. 離散程度

    離散程度用來描述資料彼此之間的差異，以及資料距離中心位置的分散程度。

    全距（Range）

    全距是資料最大值與最小值之差。

    全距 = 最大值 − 最小值 
    scores <- c(60, 65, 70, 75, 80, 90)

max(scores) - min(scores)

    也可以先使用 range() 找出最小值與最大值： 

    range(scores)

diff(range(scores))

    變異數（Variance）

    變異數衡量資料與平均數之間的平方差距。 R 的 var() 預設計算的是樣本變異數。 

    # 樣本變異數
var(scores)

    樣本變異數的分母為 n - 1。若資料代表完整母體， 母體變異數的分母應使用 n。 

    # 母體變異數
population_variance <- mean(
  (scores - mean(scores))^2
)

population_variance

    標準差（Standard Deviation）

    標準差是變異數的平方根，單位與原始資料相同，因此通常比變異數更容易解讀。 

    # 樣本標準差
sd(scores)
    # 母體標準差
population_sd <- sqrt(
  mean((scores - mean(scores))^2)
)

population_sd
    R 的 var() 與 sd() 預設使用樣本公式，分母為 n − 1。

    5. MAD

    MAD 通常指中位數絕對偏差 （Median Absolute Deviation），用來衡量資料與中位數之間的偏離程度。

    由於 MAD 以中位數為中心，因此比標準差更不容易受到極端值影響。 

    scores <- c(60, 65, 70, 75, 80, 90, 200)

mad(scores)

    R 的 mad() 預設會乘上一個調整常數，使結果在常態分布下能與標準差比較。 若希望計算未調整的中位數絕對偏差，可以設定： 

    mad(
  scores,
  constant = 1
)

    也可以自行計算： 

    median(
  abs(scores - median(scores))
)
    當資料中可能存在離群值時，MAD 通常比標準差更穩健。

    6. 變異係數（Coefficient of Variation）

    變異係數簡稱 CV，是標準差與平均數的比值，用來衡量相對分散程度。

    CV = 標準差 ÷ 平均數

    CV 沒有單位，因此可以用來比較單位不同或平均水準不同的資料。

    樣本變異係數

    scores <- c(60, 65, 70, 75, 80, 90)

sample_cv <- sd(scores) / mean(scores)

sample_cv

    若希望以百分比呈現：

    sample_cv_percent <- (
  sd(scores) / mean(scores)
) * 100

sample_cv_percent

    母體變異係數

    population_sd <- sqrt(
  mean((scores - mean(scores))^2)
)

population_cv <- population_sd / mean(scores)

population_cv
    population_cv_percent <- population_cv * 100

population_cv_percent
    CV 通常適用於平均數為正且具有比例尺度的資料。當平均數接近 0 時，CV 可能非常大或失去解釋意義。

    7. 共變異數（Covariance）

    共變異數用來描述兩個變數是否傾向一起變動。

    • 共變異數大於 0：兩個變數大致同方向變動。
    • 共變異數小於 0：兩個變數大致反方向變動。
    • 共變異數接近 0：兩者可能沒有明顯的線性共同變動。

    樣本共變異數

    R 的 cov() 預設計算樣本共變異數。 

    x <- c(2, 3, 4, 5, 6, 7, 8, 9)

y <- c(55, 60, 65, 68, 72, 78, 82, 88)

cov(x, y)

    母體共變異數

    population_covariance <- mean(
  (x - mean(x)) *
  (y - mean(y))
)

population_covariance
    共變異數會受到資料單位影響，因此通常不適合直接比較不同變數組合的關係強弱。

    8. 相關係數（Correlation）

    相關係數將共變異數標準化，數值介於 −1 與 1 之間。

    • 接近 1：強烈正線性關係。
    • 接近 −1：強烈負線性關係。
    • 接近 0：線性關係較弱。
    cor(x, y)

    cor() 預設計算 Pearson 相關係數： 

    cor(
  x,
  y,
  method = "pearson"
)

    若資料為順序尺度、存在離群值，或關係並非線性但大致單調， 可以考慮 Spearman 等級相關係數： 

    cor(
  x,
  y,
  method = "spearman"
)
    相關不代表因果。即使兩個變數高度相關，也不能只憑相關係數認定其中一個變數造成另一個變數改變。

    9. 相關矩陣

    當資料中有多個數值變數時，可以使用 cor() 一次計算所有變數兩兩之間的相關係數。 

    df <- data.frame(
  study_hours = c(2, 3, 4, 5, 6, 7, 8, 9),
  attendance = c(65, 70, 72, 75, 80, 85, 88, 92),
  score = c(55, 60, 65, 68, 72, 78, 82, 88)
)

cor(df)

    若資料中包含缺失值，可以使用： 

    cor(
  df,
  use = "complete.obs"
)

    10. Sharpe Ratio

    Sharpe Ratio 用來衡量每承擔一單位風險所獲得的超額報酬， 常見於投資績效與金融風險分析。

    Sharpe Ratio =（平均報酬率 − 無風險報酬率）÷ 報酬率標準差 
    returns <- c(
  0.03, 0.02, -0.01, 0.04,
  0.015, 0.025, -0.005, 0.035
)

risk_free_rate <- 0.002

sharpe_ratio <- (
  mean(returns) - risk_free_rate
) / sd(returns)

sharpe_ratio

    Sharpe Ratio 越高，通常表示相對於承擔的風險，獲得的超額報酬越高。

    報酬率與無風險利率必須使用相同期間。例如月報酬率應搭配月無風險利率，不能直接混用年利率。

    若使用月報酬率，並希望將 Sharpe Ratio 年化，可以乘以 sqrt(12)： 

    annualized_sharpe <- sharpe_ratio * sqrt(12)

annualized_sharpe

    11. 快速查看敘述統計

    R 的 summary() 可以快速顯示最小值、第一四分位數、 中位數、平均數、第三四分位數與最大值。 

    scores <- c(60, 65, 70, 75, 80, 90)

summary(scores)

    若資料為 Data Frame，也可以一次查看多個欄位： 

    summary(df)

    12. 散佈圖與線性趨勢線

    散佈圖可以呈現兩個數值變數之間的關係。 若點大致沿著直線方向分布，表示兩個變數可能存在線性關係。

    使用 Base R

    x <- c(2, 3, 4, 5, 6, 7, 8, 9)

y <- c(55, 60, 65, 68, 72, 78, 82, 88)

plot(
  x,
  y,
  main = "Study Hours and Scores",
  xlab = "Study Hours",
  ylab = "Score",
  pch = 19,
  col = "#6FA8DC"
)

model <- lm(y ~ x)

abline(
  model,
  col = "#E07A5F",
  lwd = 2
)

    lm(y ~ x) 建立簡單線性迴歸模型， abline() 則將迴歸直線加入散佈圖。

    使用 ggplot2

    library(ggplot2)

df <- data.frame(
  study_hours = c(2, 3, 4, 5, 6, 7, 8, 9),
  score = c(55, 60, 65, 68, 72, 78, 82, 88)
)

ggplot(
  df,
  aes(
    x = study_hours,
    y = score
  )
) +
  geom_point(
    size = 3,
    color = "#6FA8DC"
  ) +
  geom_smooth(
    method = "lm",
    se = TRUE,
    color = "#E07A5F",
    fill = "#F6D6CC"
  ) +
  labs(
    title = "Study Hours and Scores",
    subtitle = "Linear regression trend line",
    x = "Study Hours",
    y = "Score"
  ) +
  theme_minimal()

    在 geom_smooth() 中：

    • method = "lm"：使用線性迴歸模型。
    • se = TRUE：顯示信賴區間。
    • color：設定趨勢線顏色。
    • fill：設定信賴區間顏色。
    趨勢線可以幫助判斷變數之間的大致方向，但仍應搭配相關係數、迴歸結果與資料背景進行解讀。

    13. 指標比較

    集中趨勢： 平均數、中位數、眾數、幾何平均、加權平均

    絕對離散程度： 全距、變異數、標準差、MAD

    相對離散程度： 變異係數 CV

    兩變數共同變動： 共變異數

    線性關係強度： 相關係數

    風險調整後報酬： Sharpe Ratio

    14. 常見錯誤

    • 將 R 的 mode() 誤認為統計上的眾數。
    • 未區分樣本標準差與母體標準差。
    • 平均數接近 0 時仍直接使用 CV。
    • 將相關關係直接解釋為因果關係。
    • 比較 Sharpe Ratio 時使用不同期間的報酬率。
    • 看到相關係數接近 0，就認定兩個變數完全沒有任何關係。
    • 只看趨勢線，沒有檢查離群值或非線性關係。

    練習題

    1. 建立一組數值資料，計算平均數、中位數與眾數。
    2. 計算該資料的全距、樣本變異數與樣本標準差。
    3. 自行計算母體變異數與母體標準差。
    4. 計算樣本 CV 與母體 CV，並以百分比呈現。
    5. 使用兩組資料計算共變異數與 Pearson 相關係數。
    6. 使用一組月報酬率計算 Sharpe Ratio。
    7. 使用 ggplot2 繪製散佈圖並加入線性趨勢線。
    8. 說明為什麼相關係數高不一定代表存在因果關係。

    本章重點整理

    • 平均數、中位數與眾數用來描述資料的中心位置。
    • 幾何平均適合成長倍數，加權平均適合不同重要程度的資料。
    • 全距、變異數、標準差與 MAD 可描述資料的離散程度。
    • R 的 var() 與 sd() 預設使用樣本公式。
    • CV 可比較不同尺度資料的相對分散程度。
    • 共變異數反映共同變動方向，但會受到單位影響。
    • 相關係數介於 −1 與 1 之間，但相關不代表因果。
    • Sharpe Ratio 用來衡量每單位風險所帶來的超額報酬。
    • 散佈圖搭配線性趨勢線，可協助觀察兩個變數的線性關係。
