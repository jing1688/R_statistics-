    Chapter 01

    RStudio 與 Posit Cloud 基礎

    認識 R、RStudio、Posit Cloud 與基本語法

    1. 什麼是 R？

    R 是專為統計分析、資料視覺化與資料科學設計的程式語言。

    • 免費且開源
    • 內建大量統計函數
    • 擁有完整的 CRAN 套件生態系
    • 適合研究與商業資料分析

    2. RStudio 與 Posit Cloud

    RStudio 是常用的 R 開發環境；Posit Cloud 則可直接在瀏覽器執行 R。

    RStudio 介面

    • Source：撰寫與保存程式碼
    • Console：即時執行指令
    • Environment：查看物件與資料
    • Files / Plots / Packages / Help：管理檔案、圖表、套件與說明
    不想安裝軟體時，可以前往 Posit Cloud。

    3. R 基本語法

    變數與指定

    name <- "Alice"
age <- 20
height <- 165.5
is_student <- TRUE

print(name)

    R 最常使用 <- 指定變數。

    向量

    scores <- c(85, 90, 78, 92)

scores[1]        # R 的索引從 1 開始
mean(scores)
length(scores)
    注意：R 的索引從 1 開始，和 Python 從 0 開始不同。

    條件判斷

    score <- 92

if (score >= 90) {
  print("優異")
} else {
  print("普通")
}

    迴圈

    for (score in scores) {
  print(score)
}

    函式

    add <- function(a, b) {
  return(a + b)
}

add(3, 5)

    4. 常見資料結構

    List

    student <- list(
  name = "Alice",
  age = 20,
  score = 90
)

student$name

    Data Frame

    students <- data.frame(
  name = c("Alice", "Bob", "Carol"),
  score = c(90, 82, 95)
)

students

    5. 安裝與載入套件

    # 第一次使用時安裝
install.packages("tidyverse")

# 每次開啟 R 後載入
library(tidyverse)
    套件通常只需安裝一次，但每次重新開啟 R 都要使用 library() 載入。

    本章重點整理

    • R 適合統計分析與視覺化
    • RStudio 與 Posit Cloud 都能執行 R
    • R 常用 <- 指定變數，索引從 1 開始
    • 常見結構包含向量、List 與 Data Frame

    2. 建立包含姓名、年齡與成績的資料框。
    3. 撰寫一個回傳兩數較大值的函式。
