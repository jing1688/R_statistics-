    1. Getting Started with R

    R 是一款專為統計分析、資料處理與資料視覺化設計的免費軟體環境，同時也是一門程式語言。它融合統計套件的便利性與程式設計的彈性，廣泛應用於資料分析、機器學習、商業分析與科學研究。

    R 不只能執行平均數、迴歸分析與假設檢定，也能整理資料、建立自動化分析流程，並輸出高品質圖表。

    R 語言的主要特色

    • 容易取得：R 可以免費下載與使用。
    • 自由性高：使用者可以自行撰寫函式與套件。
    • 功能完整：涵蓋統計分析、資料整理、機器學習與視覺化。
    • 套件豐富：可從 CRAN 安裝套件，依需求擴充功能。
    • 文件完整：可透過 Help 查詢函式、參數及使用範例。
    • 高階語言：可以較少程式碼完成複雜分析。
    • 繪圖能力強：可使用內建繪圖或 ggplot2 製作高品質圖表。

    1.1 安裝與 R 語言入門

    安裝 R

    1. 前往 CRAN 官方網站。
    2. 依作業系統選擇 Windows、macOS 或 Linux。
    3. 下載並安裝最新版本的 R。

    安裝 RStudio

    RStudio 是常用的 R 整合開發環境，可集中管理程式碼、Console、變數、圖表、套件與說明文件。

    1. 先完成 R 的安裝。
    2. 前往 RStudio Desktop 下載。
    3. 安裝後開啟 RStudio，確認 Console 可以正常執行指令。

    第一個 R 指令

    # 在 Console 輸入
1 + 1

# 執行結果
# [1] 2

    輸出前的 [1] 代表該行第一個顯示的元素，在完整結果中位於第 1 個位置。

    教學檔案：目前尚未提供實際 PDF。上傳 PDF 後，再把這裡改成正式下載連結即可。

    1.2 設定工作目錄（Working Directory）

    工作目錄是 R 預設讀取與儲存檔案的位置。使用相對路徑時，R 會從目前工作目錄尋找檔案。

    查看目前工作目錄

    getwd()

# 範例輸出
# [1] "D:/college information/R_statistics"

    設定工作目錄

    setwd("D:/college information/R_statistics")
getwd()

# 範例輸出
# [1] "D:/college information/R_statistics"
    Windows 路徑提醒：建議使用正斜線 /，不要直接使用單一反斜線。

    使用相對路徑

    # data 資料夾位於目前專案中
file_path <- "data/student_scores.csv"

    相對路徑較容易分享，也能降低換電腦後路徑失效的機率。實務上建議使用 RStudio Project 管理每一份作業或研究。

    1.3 基本操作

    基本運算

    3 + 5      # 加法
10 - 4     # 減法
6 * 7      # 乘法
20 / 4     # 除法
2 ^ 3      # 次方
10 %% 3    # 餘數
10 %/% 3   # 整數除法

    指定變數

    x <- 10
y <- 20

x + y
# [1] 30

    R 可使用 <- 或 = 指定變數，但一般 R 程式較常使用 <-。

    建立向量

    # 定義向量 x 為 2 到 9
x <- 2:9

x
# [1] 2 3 4 5 6 7 8 9

# 另一種建立向量的方法
z <- c(1, 2, 3, 4)

    向量化運算

    x <- 2:9

x + 1
# [1] 3 4 5 6 7 8 9 10

x * 5
# [1] 10 15 20 25 30 35 40 45

y <- 2 * x

    R 的運算可以直接作用在向量中的每個元素，通常不需要逐筆撰寫迴圈。

    向量型態一致性

    x <- c(3, "abc", 4)

x
# [1] "3"   "abc" "4"

    原子向量只能保存同一種基本型態。數值與文字混合時，R 會把數值轉成文字。

    向量索引

    x <- c(3, 5, 9)

x[2]
# [1] 5

x[c(1, 3)]
# [1] 3 9

x[1:2]
# [1] 3 5
    重要：R 的索引從 1 開始，和 Python 從 0 開始不同。

    邏輯索引

    scores <- c(65, 80, 92, 70, 88)

scores[scores >= 80]
# [1] 80 92 88

    常用向量函式

    scores <- c(65, 80, 92, 70, 88)

length(scores)
sum(scores)
mean(scores)
min(scores)
max(scores)
sort(scores)

    1.4 資料型態

    可以使用 class()、typeof() 或 str() 查詢資料型態；View() 則適合表格資料。

    1.4.1 Character：文字字串

    x <- "Hello world."

x
# [1] "Hello world."

class(x)
# [1] "character"

    1.4.2 Numeric：數值型態

    x <- 100

class(x)
# [1] "numeric"

x + 5
x - 5
x * 5
x / 3
x %% 3

v <- c(10, 11, 12, 11)
sum(v)
# [1] 44

    1.4.3 Integer：整數型態

    x <- 3L
class(x)
# [1] "integer"

y <- as.integer(3.5)
y
# [1] 3
    注意：as.integer() 會捨去小數，不是四捨五入；四捨五入請用 round()。

    1.4.4 Complex：複數型態

    z <- 3 + 2i

z
# [1] 3+2i

class(z)
# [1] "complex"

Re(z)   # 實部
Im(z)   # 虛部
Mod(z)  # 模長

    1.4.5 Logical：邏輯值

    x <- TRUE
y <- FALSE

class(x)
# [1] "logical"

score <- 85
score >= 60
# [1] TRUE

    雖然可以使用 T 與 F，但建議寫完整的 TRUE、FALSE，避免被同名變數覆蓋。

    缺失值 NA

    scores <- c(80, 90, NA, 75)

mean(scores)
# [1] NA

mean(scores, na.rm = TRUE)
# [1] 81.66667

    1.5 資料型態的轉換

    可使用 as.資料型態() 將物件轉成不同型態。轉換前應確認內容是否合理，否則可能產生 NA。

    轉成 Character

    x <- 3
x <- as.character(x)

class(x)
# [1] "character"

    轉成 Numeric

    x <- "3.14"
x <- as.numeric(x)

class(x)
# [1] "numeric"

    無法轉換的內容

    x <- "apple"
as.numeric(x)

# [1] NA
# Warning message:
# NAs introduced by coercion

    轉成 Factor

    group <- c("A", "B", "A", "C")
group <- as.factor(group)

class(group)
# [1] "factor"

levels(group)
# [1] "A" "B" "C"
    • as.character()：轉成文字。
    • as.numeric()：轉成數值。
    • as.integer()：轉成整數。
    • as.logical()：轉成邏輯值。
    • as.factor()：轉成類別資料。
    • as.Date()：轉成日期。

    1.6 其他小功能

    註解

    # 井字號後面的內容不會執行
x <- 10

x + 5   # 也可以寫在指令後方

    查詢說明文件

    ?mean
help(mean)

??ggplot

args(mean)
example(mean)

    查看與移除物件

    ls()            # 查看目前物件
rm(x)           # 刪除單一物件
rm(list = ls()) # 刪除全部物件
    小心：rm(list = ls()) 會清空目前環境，執行前先確認重要結果已儲存。

    R 使用建議

    • 避免中文路徑與檔名：優先使用英文，以降低編碼相容性問題。
    • 先檢查資料型態：讀入資料後先使用 class()、str() 或 summary()。
    • 注意大小寫：score 與 Score 是不同物件。
    • 命名要有意義：student_score 通常比 x1 更容易理解。
    • Warning 不是 Error：Warning 代表程式可能已執行，但結果可能有潛在問題；仍應閱讀訊息，而不是全部忽略。

    本章重點整理

    • R 是免費且適合統計分析、資料處理與視覺化的程式語言。
    • getwd() 與 setwd() 可查看及設定工作目錄。
    • R 支援向量化運算，且索引從 1 開始。
    • 常見型態包含 character、numeric、integer、complex 與 logical。
    • 可使用 as.*() 系列函式轉換資料型態。
    • 轉換失敗可能產生 NA 與 warning。

    練習題

    1. 使用 getwd() 查看工作目錄，並說明它和檔案讀取的關係。
    2. 建立 x <- 1:10，計算每個元素的平方。
    3. 從 c(65, 80, 92, 70, 88) 取出大於等於 80 的數值。
    4. 建立 character、numeric、integer、complex 與 logical 物件，並用 class() 檢查。
    5. 將字串 "100" 轉成 numeric，再加上 20。
    6. 將 "apple" 轉成 numeric，並解釋 warning。
    7. 使用 ?mean 查詢 na.rm 參數。
