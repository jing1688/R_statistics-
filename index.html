    R × Statistics

    用 R 語言學統計學

    從 RStudio、資料整理與視覺化開始，逐步進入機率、抽樣、統計推論、假設檢定與迴歸分析。

    課程介紹

    純 HTML、CSS 與 JavaScript，可直接部署到 GitHub Pages。

    適合對象

    想用 R 入門資料分析、統計學與視覺化的學習者。

    建議環境

    R、RStudio 或 Posit Cloud，以及 tidyverse、readxl、ggplot2。
